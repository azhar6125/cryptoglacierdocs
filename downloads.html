What's included

Once you download and extract the Glacier zip file, you'll get:


├── CryptoGlacier.pdf
├── LICENSE
├── README.md
├── SHA256SUMS
├── SHA256SUMS.sig
├── mnemonic_entropy.py
├── package-lock.json
├── package.json
└── setup.js
└── ether.js

This includes the CryptoGlacier scripts and PDF file with the CryptoGlacier protocol. It also includes checksum and signature files for you to validate if the package you got can be trusted. Learn how to use these files.
